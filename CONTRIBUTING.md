## How to Contribute

### 1. Found a bug?
- Check the [Issues](https://github.com/Zypher0903/FluxOS/issues) tab first.
- If it doesn't exist, open a new issue with:
  - Clear title
  - Steps to reproduce
  - Expected vs actual behavior
  - Screenshots (if GUI-related)

### 2. Want to fix something or add a feature?
